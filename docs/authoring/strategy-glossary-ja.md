# 戦略スキル 日本語用語集

分析レイヤーの 15 スキル（`acrosstudio-intake` ＋ 13 の `strat-*` ＋ `acrosstudio-engagement`）は、すべて**日本語を第一言語**として記述します。本用語集は、スキル間で用語と見出しを揃えるための正本です。新しいスキルを日本語化するときは、必ずこの用語集の表記に合わせてください。

スキルの識別子（`name`、ディレクトリ名、`engagement/NN-*.md` のファイル名、各プラットフォームのマニフェスト）は**変更しません**。日本語化するのは自然言語のコンテンツだけです。これは `acrosstudio-pptx`（`name` は英語、本文・description は日本語）と同じ方針です。

## 文体

すべての戦略スキルは、`skills/acrosstudio-pptx/reference/content_patterns.md` の**文体・禁止語**に従います。要点は次のとおりです。

- 機械翻訳・直訳にしない。日本語として自然に読めるよう書き直す。
- 顧客企業は「貴社」、自社は「弊社」で統一する。
- 1 文 50 字以内を目安にし、断定や誇大表現（「絶対に」「劇的に」など）を避ける。
- 推測は「想定」「ご参考」と明示し、数値を勝手に作らない。不明な数値は確認事項として残す。
- `PoC`・`アジャイル`・`エンドツーエンド` などの禁止語は置換する。一方で `ROI`・`KPI` のように日本のビジネスでそのまま使われる語はそのまま残す。

## セクション見出し

スキル本文の節構成は、英語テンプレートの 6 節を次のとおり日本語化します。

| English | 日本語 |
| --- | --- |
| Overview | 概要 |
| When to use | 使用する場面 |
| Inputs | 必要な情報 |
| Method | 進め方 |
| Output | 成果物 |
| Example | 例 |

## 出力契約の見出し

各スキルが生成する分析（成果物）は、次の 2 つの見出しで締めくくります。順序も固定です。

| English | 日本語 |
| --- | --- |
| Key takeaways | `## 要点` |
| Open questions | `## 確認事項` |

`## 確認事項` は、「数値を勝手に作らず、確認のために残す」という弊社の方針にそのまま合います。確定していない数値・前提・判断は、ここに列挙します。

成果物の冒頭は、次の形のタイトルで始めます。`<フレームワーク>` は下表の日本語タイトル、`<対象>` は分析対象の案件テーマです。

```markdown
# <フレームワーク> — <対象>
```
