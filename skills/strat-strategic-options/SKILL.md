# 戦略オプション

## 概要

戦略オプション分析は、意思決定を「真の選択」として扱う手法です。
一つの答えに早期に絞り込む前に、「現状維持」を含む相互排他的・網羅的な選択肢を列挙します。
その上で、明示的な評価基準にもとづき比較を行います。
早期に単一案に傾くことで、より良い選択肢を見落とすリスクがあります。
構造化された比較は、意思決定の客観性を保ちます。

## 使用する場面

- 戦略的な意思決定が確定しておらず、複数の進め方が現実的に考えられる場合。
- 貴社または担当チームがすでに特定の案を有力視しており、代替案を明示することでその妥当性を検証したい場合。
- 意思決定者が経営幹部や取締役会に対して、明確な根拠とともに選択理由を説明する必要がある場合。
- **使用しない場面：** 方針がすでに確定し、実行計画の検討段階にある場合は `strat-transformation-roadmap` または `strat-initiative-prioritizer` を使用してください。また、法規制・コスト・スケジュールの制約により現実的な選択肢が事実上一つに絞られている場合も、このスキルは適していません。架空の選択肢を並べることは分析の負担になるだけです。

## 必要な情報

- 検討すべき意思決定を、時間軸を含む具体的な問いの形で表現したもの（例：「今後 12 ヶ月以内に国内新カテゴリへ参入すべきか」）。
- 貴社にとっての評価基準——「良い選択」の定義（例：収益化までの期間、必要資本、リスク許容度、戦略適合性）。評価基準が明確でない場合は、分析を進める前にご確認ください。
- 現状の立ち位置、制約条件、利用可能なリソース、すでに検討中の選択肢などの関連情報。予算・スケジュール・市場規模などの数値が不明な場合は、担当の方に確認を求めます。数値を勝手に作ることはしません。
- 特定の選択肢を自動的に除外する絶対条件（もし存在する場合）。

## 進め方

1. **意思決定と評価基準を確定する。** 意思決定を時間軸を含む一つの問いとして記述します。選択肢を比較する評価基準を 3〜5 個列挙します。貴社が明確な優先度を示している場合のみ重み付けを行い、そうでない場合は等ウェイトとし、その選択自体を確認事項として残します。

2. **「現状維持」案を含む MECE な選択肢を生成する。** 相互排他的（コアロジックが重複しない）かつ網羅的（現実的な選択空間を網羅する）な選択肢を作成します。ベースラインとして「何もしない」または「現状維持」案を必ず含めます。選択肢の数は 3〜5 案が適切です。それ以下では検討が不十分になりやすく、それ以上は管理が難しくなります。

3. **各選択肢を評価基準で評価する。** 各評価基準に対して各選択肢のパフォーマンスを評価します。貴社から定量データが提供されていない限り、定性評価（高・中・低）を用います。各評価の根拠を明示し、貴社が内容を確認・異議申立てできるようにします。

4. **トレードオフと可逆性を明らかにする。** 選択肢が相反する方向を向いている箇所——スピード対コスト、管理対柔軟性、上昇余地対リスク——を特定します。各選択肢が「後から修正できる（可逆的）」か「将来の選択肢を閉ざす（非可逆的）」かを明記します。評価基準の点数が接近している場合、可逆性は重要な副次的判断材料になります。

5. **推奨案を根拠とともに示す。** 優先する選択肢を明示します。証拠が結論を一つに絞れない場合は、特定の不確実性が解消された場合の条件付き推奨とします。推奨の根拠を 2〜4 文で示し、推奨案に対する最も強い反論も記載します。

## 成果物

本スキルは、以下の構成で単独のマークダウン分析を作成します。

```
# 戦略オプション — <対象>

## 意思決定と評価基準
## 選択肢
## 評価
## トレードオフ
## 推奨案
## 要点
## 確認事項
```

分析は `# 戦略オプション — <対象>` というタイトルで始まります。`<対象>` は分析対象の意思決定テーマを指します（例：「戦略オプション — ヤマブキフーズ 低価格競合への対応方針」）。本文は上記の節構成を順に展開します。分析は必ず `## 要点`（意思決定者が持ち帰るべき 2〜4 点の重要な結論）と `## 確認事項`（推奨案を確定する前に貴社への確認が必要な事項・不足データ・次の意思決定）で締めくくります。数値を勝手に作ることはせず、不明な情報はすべて確認事項として残します。

## 例

**例 — 架空の食品メーカー「ヤマブキフーズ株式会社」（例示 / 確認前）**

*意思決定：ヤマブキフーズは、低価格帯の新規参入競合に対してどのように対応すべきか。*

**評価基準：** 収益率の維持、対応スピード、必要資本、可逆性。

**生成された選択肢：**
1. 現状維持 ——現行の価格を維持し、競合の市場定着度を観察する。
2. 価格防衛 ——競合が攻勢をかけているアカウントで選択的に価格を合わせる。
3. 差別化強化 ——品質・素材・サービス力への投資を加速し、価格プレミアムを正当化する。
4. 買収・提携 ——新規参入者との合弁または買収について打診する。

**評価（例示 / 確認前）：**
- 選択肢 3 は収益率の維持と戦略適合性の評価が最も高くなりますが、必要な投資資本は貴社でご確認ください。
- 選択肢 2 は短期的な販売量を守りますが、収益率と可逆性の評価は低くなります。一度価格を下げると、回復は容易ではありません。
- 選択肢 4 はインパクトが大きい反面、不確実性も高く、財務的な事前評価が必要です。確認事項として残します。

**推奨案：** 選択肢 3（差別化強化）を優先して進め、差別化施策が軌道に乗るまでの間は、離反リスクの高い上位 3〜5 アカウントに限り選択肢 2（価格防衛）を補助的に適用します。

**要点**
- 低価格の新規参入者が主に脅かすのは汎用品カテゴリーであり、高付加価値ラインへの影響は限定的と想定されます（確認前）。
- 差別化強化は、現状防衛ではなく中長期の競争優位を構築する唯一の選択肢です。

**確認事項**
- 差別化施策への投資に充てられる予算規模はどの程度ですか？
- 価格防衛が必要な上位アカウントを 3〜5 社特定できますか？
- 選択肢 4（提携・買収）について、新規参入者との予備的な対話を検討しますか？
