# 事業ポートフォリオ評価

## 概要

事業ポートフォリオ評価は、有限な経営資源（資本・人員・経営の注意力）を、複数の事業ユニット（製品・事業部門・地域市場・施策）にどのように配分するかを決めるためのフレームワークです。

各ユニットを「事業機会の魅力度（市場の外部環境）」と「貴社が勝てる競争優位性（内部の強み）」の2軸で評価し、マトリクス上に配置します。両軸ともにスコアが高いユニットには重点投資が妥当であり、両軸ともにスコアが低いユニットは撤退・縮小の候補となります。本フレームワークの価値は、「一律に薄く広げる」のではなく「どこに集中するか」を相対的・明示的に議論する場を設けることにあります。

## 使用する場面

次のような場面でお使いください。

- 予算・人員・経営者の判断力を、複数の競合するユニットにどう振り分けるかを決める必要がある。
- ポートフォリオが自然な拡張を重ねた結果、各ユニットへの資源配分が現状のままでよいか立ち止まって見直す必要がある。
- 新たな中長期戦略を策定する際、既存事業のどれが新戦略を支えるかを確認したい。
- 推進候補の施策が実行可能な数を上回っており、絞り込んだ推進リストに整理する必要がある。

**お使いにならない方がよい場面：**

- ポートフォリオに事業ユニットが1つしかない場合は、`strat-situation-assessment` または `strat-growth-barriers` をご検討ください。
- 課題がロードマップの順序付けであり、資源配分ではない場合は、`strat-transformation-roadmap` をご検討ください。
- 特定の投資案件の採算性が問われている場合は、`strat-business-case` をご検討ください。

## 必要な情報

以下の情報が必要です。不明な項目や不確かな項目がある場合は、着手前にご確認ください。数値を勝手に補うことはいたしません。

1. **ポートフォリオ・ユニットの一覧** — 評価対象とする製品・事業部門・市場・施策の名称。
2. **魅力度の指標** — 各ユニットの市場規模・成長率・競合の強度・戦略適合度。不明なものは明示してください。
3. **競争優位性の指標** — 各ユニットの相対的市場シェアまたは受注率・ケイパビリティの強み・マージン水準・顧客継続率。
4. **資源の総量** — 配分対象となる予算・人員・その他の希少資源の合計量。
5. **戦略的優先事項** — 純粋な魅力度ロジックを上書きする非交渉条件（規制要件・フラッグシップ案件など）。

上記のうち取得できない数値は、`## 確認事項` に列挙します。数値を想定で埋めることはいたしません。

## 進め方

1. **ユニットと評価軸の確定。** ポートフォリオ・ユニットの一覧を確認します。評価軸は、魅力度（外部環境）× 競争優位性（内部の強み）を既定軸とします。ただし、成長率 × 相対シェア（BCG スタイル）や戦略適合度 × 財務リターンなど、文脈に応じた代替軸も有効です。選択した軸と、それを構成する個別指標を明示します。

2. **スコアリングとマトリクス配置。** 各ユニットを各軸でスコアリングします（例：高 / 中 / 低、または 1〜5 のスケール）。貴社が提供したデータを使用します。前提に基づくスコアにはフラグを立てます。結果を 2×2 または 3×3 のマトリクスに配置します。

3. **方針（Verdict）の付与。** マトリクス上の位置をもとに、各ユニットに経営方針を付与します。**Grow**（魅力度・競争優位性ともに高い — 投資拡大）、**Hold**（一定のパフォーマンスを維持 — 現状維持）、**Fix**（不振だが回復余地あり — 期間を定めた集中投資）、**Exit**（魅力度・競争優位性ともに低い — 売却・縮小・優先度引き下げ）の4区分を用います。

4. **資源配分の現状確認。** 現時点で各ユニットに流れている資源量を集計し、方針と対比します。Grow 判定なのに資源が不足しているユニット、Exit 判定なのに資源を過剰に消費しているユニットの乖離を特定します。

5. **配分転換案の提示。** 乖離を具体的な配分転換の提案に落とし込みます。資源の移動内容・移動量・実行に必要な意思決定を明示します。組織上の摩擦を生む移動にはフラグを立てます。

## 成果物

このスキルを適用すると、以下の構成のスタンドアローン分析を生成します。分析は冒頭タイトルで始まり、`## 要点` と `## 確認事項` で終わります。

```
# 事業ポートフォリオ評価 — <対象>

## ユニット一覧と評価軸
ポートフォリオ・ユニットをすべて列挙し、選択した評価軸（各軸を構成する個別指標を含む）を定義します。

## ポートフォリオ・マップ
各ユニットをマトリクス上に配置し、各軸のスコアを示します。表またはマトリクス形式で読みやすく整理します。前提に基づくスコアにはフラグを立てます。

## ユニット別判定
各ユニットへの Grow / Hold / Fix / Exit 判定を、1〜2 文の根拠とともに記載します。

## 資源配分の見直し
判定結果から導かれる具体的な資源移動を記載します。何を・どれだけ・どのような意思決定を経て動かすかを明示します。

## 要点
本評価で得られた最重要の結論を箇条書きで示します。

## 確認事項
貴社による確認が必要な事項、不足データ、配分転換を進める前に決定が必要な事項を列挙します。
```

## 例

**例 — 架空の国内食品メーカー（株式会社サカエフーズ）**（例示 / 確認前）

サカエフーズは4つの事業部門を持ちます。主力の「常温加工食品」、伸長中の「冷凍惣菜」、成熟市場の「調味料・ソース類」、立ち上げ期の「植物性代替タンパク」です。CFO は翌年度の設備投資と開発予算の配分を決定する必要があります。

**選択した評価軸：** 市場魅力度（市場成長率 + マージンプール）× 競争優位性（相対シェア + 独自製法の深度）

| 事業ユニット | 市場魅力度 | 競争優位性 | 判定 |
|---|---|---|---|
| 常温加工食品 | 中 | 高 | Hold |
| 冷凍惣菜 | 高 | 高 | Grow |
| 調味料・ソース類 | 低 | 中 | Fix または Exit |
| 植物性代替タンパク | 高 | 低 | Fix（ケイパビリティ構築） |

**配分転換のシグナル（例示）：** 冷凍惣菜はマトリクス上の位置に対して資源が不足しています。調味料・ソース類は販売人員を過剰に消費しており、その一部を冷凍惣菜に移すことでより高いリターンが期待されます。植物性代替タンパクは、追加投資の前にケイパビリティ面での判断が必要です。

**確認事項（本例からの例示）：**
- 冷凍惣菜における相対的市場シェアは、上位2社対比でどの程度か。（確認前）
- 調味料・ソース類は2年以内に黒字化できるか。それとも今期中に撤退の判断をするか。（確認前）
- 植物性代替タンパクのケイパビリティ投資について、経営会議での方針決定は済んでいるか。（確認前）
