# 現状分析

## 概要

現状分析は、事実に基づいて現在の状態を体系的に整理する診断フレームワークです。市場・事業・オペレーション・組織という四つの領域について客観的な事実を収集し、戦略的な意思決定の前提として整理します。

事実の共有なしに進める意思決定は、関係者それぞれが持つ異なるナラティブによって行き詰まりやすくなります。本フレームワークは、その分散したナラティブを一つの事実ベースの共通認識に置き換えることを目的とします。

## 使用する場面

- エンゲージメント開始時に、チームが選択肢や推奨案の議論を始める前の共通ベースラインが必要なとき。
- 意思決定者が現状に対して異なる見解を持ち、事実認識のずれから議論が前に進まなくなっているとき。
- 過去の分析やスライドが状況を記述してはいるが、確認済みの事実と仮定・推論を分けて整理できていないとき。
- **使用しない場面：** 既に合意された信頼性の高い現状ベースラインが存在する場合は、本フレームワークを省略し、次の分析スキル（`strat-growth-barriers`・`strat-strategic-options` など）に直接進む。また、貴社がすでに問題の本質を把握していて推奨案を求めている場合も、現状分析で判断を先延ばしすることは避ける。

## 必要な情報

本フレームワークを適用する前に、以下の情報を収集する。不足している項目がある場合は、進める前にご担当者様に確認を求める。数値を勝手に補うことはしない。

- **分析の核心となる問い：** この現状分析が答えるべき意思決定・課題は何か。
- **市場に関する事実：** 市場規模・成長率・主要セグメント・最近の変化。数値はご確認いただく。
- **財務に関する事実：** 売上・利益率・成長軌跡・競合との相対的な実績。実績値をご提供いただく。
- **オペレーションに関する事実：** 生産能力・コスト構造・主要プロセス・既知のボトルネック。
- **組織に関する事実：** リーダーシップ・ケイパビリティ・組織風土のシグナル・最近の変化。
- **既知の未知：** チームが把握できていないと認識している項目。推定せず、確認事項として列挙する。

## 進め方

1. **核心となる問いを設定する。** この現状分析が答えようとしていることを一文で表す（例：「なぜ売上成長が鈍化し、何を変えなければならないのか」）。以降のすべてのステップで、この問いとの関連性を判断軸とする。

2. **四つの領域にわたって事実を収集する。** 市場・財務・オペレーション・組織の各領域について確認済みのデータを集める。各事実には出所と確信度を付記し、検証済みの数値ではなく推定・仮定に過ぎないものには明示的にフラグを立てる。

3. **分析レンズを通じて事実を構造化する。** 状況に応じたレンズを選択する。市場に向き合う問いには 3C（Company・Customers・Competitors）、内部能力の問いには 7S（Strategy・Structure・Systems・Staff・Skills・Style・Shared values）、オペレーション上の問いにはバリューチェーンの視点を用いる。各事実を該当するカテゴリに配置する。事実が少ないカテゴリは、まだ十分に探索されていない領域を示している。

4. **事実と推論を区別する。** 構造化した内容を見直し、各項目に「F（確認済みの事実）」「I（事実から導いた推論）」「B（チームが持つ信念・前提）」のタグを付ける。推論は有効である。ただし、何を観察しているのか、何を結論付けているのかを意思決定者が区別できるよう、ラベルを明示する。

5. **核心課題を統合する。** 事実・推論・構造的な整理が全体として示していることを、2〜4 文で表現する。この文章が現状分析の出力となり、次の分析ステップへの直接的なインプットとなる。

## 成果物

本スキルを適用すると、以下の構成を持つ独立したマークダウン形式の分析を生成する。分析は「`# 現状分析 — <対象>`」というタイトルで始まり、下記の各セクションを経て、最後に `## 要点` と `## 確認事項` をこの順序で締めくくる。

```
# 現状分析 — <対象>

## 背景と目的
エンゲージメントの問いと現状分析の対象範囲。

## 確認済みの事実
領域（市場・財務・オペレーション・組織）ごとに整理した確認済みの事実。
各事実には出所と確信度を付記する。

## 構造的な整理
選択したレンズ（3C・7S・バリューチェーン・その他）で事実を整理した内容。
各項目に「事実（F）・推論（I）・前提（B）」のタグを付ける。

## 核心課題
エビデンスが示す中心的な問題を 2〜4 文で統合した内容。

## 要点
読者が次のステップへ持ち越すべき最も重要な結論。

## 確認事項
分析を確定させる前に、貴社への確認・追加データの収集・
フォローアップの判断が必要な項目。
```

数値を勝手に補うことはしない。入手できない数値はすべて確認事項として記載する。

## 例

**例 — 架空の消費財メーカー「東和フーズ株式会社」（例示 / 確認前）**

*核心となる問い：* なぜ東和フーズの国内売上成長が、過去 2 年間にわたりカテゴリ平均を下回っているのか。

*現状分析スナップショット：*

**確認済みの事実（抜粋）**
- 市場：国内スナック菓子カテゴリは昨年 6% 成長（例示 / 確認前、業界レポート、確信度：高）。
- 財務：東和フーズの売上成長率は昨年 1.2%、前年 0.8%（例示 / 確認前、監査済み決算、確信度：高）。
- オペレーション：主力製造ラインの稼働率は 94%（例示 / 確認前）。大手量販店 1 社から 2 SKU の取り扱いが 8 ヶ月前に廃止された（販売データ、確信度：高）。
- 組織：過去 18 ヶ月で地区営業マネージャーが 3 名退職し、補充は一部のみ完了（例示 / 確認前、人事記録、確信度：中）。

**構造的な整理 — 3C（抜粋）**
- 自社（F）：棚シェアは取り扱い廃止を受けた量販店で 18% から 14% に低下（例示 / 確認前）。他チャネルでの代替獲得は未確認（I）。
- 顧客（I）：価格弾力性データは未入手。対象期間中に価格変更があったかどうか、貴社にご確認いただく必要がある。
- 競合（F）：プレミアム帯に新規参入者 2 社が同じチャネルで配荷を開始。

**核心課題：** 東和フーズは、最大の販売チャネルで配荷を失っているタイミングと同時に、自社が参入していないプレミアム帯に競合が進出している。カテゴリ成長（6%）と自社成長（1%）の差はチャネルロスによるものとみられるが、ブランド力の問題でないとは断定できない。地区営業マネージャーの離職が実行品質を悪化させている可能性もある。いずれも仮説であり、確認が必要である。

**要点**
- 大手量販店での取り扱い廃止が、成長ギャップの最も直接的な要因とみられる。
- プレミアム帯への競合参入は構造的な脅威であるが、規模はまだ定量化されていない。

**確認事項**
- 対象期間中に価格変更は行われたか。（貴社にご確認いただく）
- 取り扱い廃止となった量販店の売上が、国内売上全体に占める割合は何%か。
- 地区営業マネージャーの離職により、営業力に測定可能な影響が出ているか。
